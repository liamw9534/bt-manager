BT-Manager
==========

BT-Manager is a library for managing bluetooth devices using Python, Bluez and DBus.
This manual is organized by firstly giving an overview of the prerequisite
configuration files that must be setup, followed by some simple worked examples of
using the API.  The reader is encouraged to check the API document and Glossary sections
to gain a deeper understanding.
